# Triple Art Gallery

# Projeto de Pátia a Pátria

## Descrição

Este projeto é uma página web responsiva para contar a história da cidade natal de profssionais da Tripleten, desenvolvida como parte do curso de Desenvolvimento Web. A página apresenta um belo design, com imagens e textos. Uma viagem cultural através da história e arte.

### Tecnologias e Técnicas Utilizadas

#### **Layout e Responsividade:**
- CSS Grid (para layouts complexos)
- Flexbox (para alinhamento de elementos)
- Design responsivo com breakpoints para mobile (320px), tablet (768px) e desktop (1280px)
- Media queries para adaptação em diferentes dispositivos

#### **Técnicas CSS Avançadas:**
- Metodologia BEM (Block Element Modifier) para organização do CSS
- Transições CSS para efeitos hover suaves
- Box-shadow para efeitos de sombra
- Gradientes CSS para efeitos visuais

#### **Ferramentas de Desenvolvimento:**
- Figma (interpretação de design e especificações)
- Git e GitHub (controle de versão e ramificações)
- GitHub Pages (hospedagem do projeto)

#### **Metodologia de Trabalho:**
- Desenvolvimento com ramificações Git (feature branches)
- Código semântico e acessível

##### Planos de Melhoria

- **Dark Mode**: Criar tema claro alternativo.
- **Layout**: Programar interface de layout para ampliação das imagens da galeria em modo tela cheia.
