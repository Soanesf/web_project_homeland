# Projeto de Pátia a Pátria

## Descrição

Este projeto é uma página web responsiva para contar a história da cidade natal de profssionais da Tripleten, desenvolvida como parte do curso de Desenvolvimento Web. A página apresenta um design retrô, em preto e branco, o qual remete a história  e a arte.
